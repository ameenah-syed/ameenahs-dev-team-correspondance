Expanded Request Summary

Raven asked Echo to identify safe remedies for tools missing from PATH, interpreter/package mismatch, npm versus pnpm ambiguity, global drift, caches, temporary directories, and native Windows builds.

Expanded Specialist Response or Execution Update

Resolve exact executables and pair package managers with their interpreter; use project-local environments and one lockfile; treat bundled runtimes as discoverable bootstrap tools; preserve caches; stop before unapproved native toolchain installation.

Evidence

  • Host and bundled Python installations differed
  • Bundled Node existed while node was absent from PATH
  • Only the pnpm shim was directly resolvable
  • No project manifest or lockfile existed at consultation time

Verification

  • Both Python installations passed pip check
  • Diagnostic prints executable identity before package state
  • Guide prohibits committing versioned cache paths

Decision Custody

Accept the recommendations into the runtime/dependency guide and read-only diagnostic.

Next Actions

  • Create a project-local environment when dependencies exist
  • Select one package manager and commit its lockfile
  • Stop before native compilation unless prerequisites are approved
